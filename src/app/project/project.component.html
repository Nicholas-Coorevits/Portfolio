Requirements Analysis

Deze opdracht is op school gemaakt voor het vak van requirements analysis gegeven door Christine Smeets


Context

Voor de Requirements Analysis projectweek kreeg mijn groep de opdracht om een applicatie te ontwerpen voor mensen met de ziekte van Alzheimer of een dissociatieve stoornis. Voor mensen met deze stoornis is het moeilijk om een overzicht te hebben van hun dag- en weekplanning. Meestal hebben ze al wel een persoonlijke agenda, maar die is vaak nog op papier ingevuld. Soms gaat het mis omdat ze sommige dingen vergeten en afspraken kunnen elkaar overlappen.

Mijn Bijdrage

In het begin werkten we vooral samen om naar de bestaande projecten te kijken en eruit te halen wat er goed uitzag. Hierna zijn we begonnen met het maken voor ons eigen project. Ik heb in het begin gewerkt aan het use case diagram. Daarna werkte ik vooral aan het projectdocument..

Bijgeleerd

Tijdens dit project heb ik geleerd hoe je het beste kunt beginnen met het analyseren van een project.

C# WPF

Deze opdracht is op school gemaakt voor het vak van DevOps gegeven door Quinten Desmyter

Context

Voor deze opdracht heb ik een windows forms gemaakt die moest werken met dapper en een micro ORM zeker. Mijn applicatie gaat de top 10 meest beluisterde liedjes van apple music in België ophalen. deze worden één voor één getoond, hier kan je zien van wie het liedje is, de naam van het liedje en de album cover. Dit windows formulier heeft 4 knoppen: "Vorige" om naar het vorige liedje te gaan. "Volgende" om naar het volgende liedje te gaan. "Opslaan" om het muzieknummer op te slaan. De opgeslagen muzieknummers worden getoond onder de knop "Opgeslagen muzieknummers" op een volgend schermformulier.

Bijgeleerd

Door dit project heb ik voor het eerst met C# leren werken. In dit project heb ik ook geleerd hoe een micro ORM en een WPF werkt.
