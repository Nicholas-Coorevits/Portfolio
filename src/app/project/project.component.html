Requirements Analysis

Deze opdracht is op school gemaakt voor het vak van requirements analysis gegeven door Christine Smeets


Context

Voor de projectweek van Requirements Analysis heeft mijn groep de opdracht gekregen om een applicatie te ontwerpen voor mensen met Alzheimer of een dissociatieve stoornis. Voor mensen met deze aandoening is het moeilijk om een overzicht te hebben van hun dag- en weekplanning. Meestal hebben ze al een persoonlijke agenda, maar deze wordt vaak nog op papier ingevuld. Soms loopt het wel eens fout omdat ze sommige dingen vergeten en afspraken met elkaar kunnen overlappen.

Mijn Bijdrage

We hebben in het begin vooral samen gewerkt voor de bestaande projecten te bekijken en hier uit te halen wat er goed uitzag. Na dit zijn we begonnen voor ons eigen project te maken. Ik heb in het begin aan het use case diagram gewerkt. Hierna ben ik vooral bezig geweest met het document van het project.

Bijgeleerd

Tijdens dit project heb ik geleerd hoe je het beste begint aan het analyseren van een project.

C# WPF

Deze opdracht is op school gemaakt voor het vak van DevOps gegeven door Quinten Desmyter

Voor deze opdracht heb ik een windows forms gemaakt die zeker met dapper en een micro ORM moest werken. Mijn applicatie gaat de top 10 meest beluisterde liedjes van apple music in België afhalen. deze worden één voor één getoond, hier kan je dan zien van wie het nummer is, de naam van het lied en de album cover. Deze windows form heef 4 knoppen: "Previous" voor naar het vorige nummer te gaan. "Next" voor naar het volgende nummer te gaan. "Save" voor het nummer op te slaan. De opgeslagen nummers komen bij de "Saved Songs" knop in een lijst te staan op een volgende windows forms.

Bijgeleerd

Door dit project ben ik voor de eerste keer met C# gewerkt. In dit project heb ik ook geleerd hoe een micro ORM en een WPF werkt
