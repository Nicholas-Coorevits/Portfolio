Requirements Analysis

Deze opdracht is op school gemaakt voor het vak van requirements analysis gegeven door Christine Smeets


Context

Voor de Requirements Analysis projectweek kreeg mijn groep de opdracht om een applicatie te ontwerpen voor mensen met de ziekte van Alzheimer of een dissociatieve stoornis. Voor mensen met deze stoornis is het moeilijk om een overzicht te hebben van hun dag- en weekplanning. Meestal hebben ze al wel een persoonlijke agenda, maar die is vaak nog op papier ingevuld. Soms gaat het mis omdat ze sommige dingen vergeten en afspraken kunnen elkaar overlappen.

Mijn Bijdrage

In het begin werkten we vooral samen om naar de bestaande projecten te kijken en eruit te halen wat er goed uitzag. Hierna zijn we begonnen met het maken voor ons eigen project. Ik heb in het begin gewerkt aan het use case diagram. Daarna werkte ik vooral aan het projectdocument.

Bijgeleerd

Tijdens dit project heb ik geleerd hoe je het beste kunt beginnen met het analyseren van een project.

C# WPF

Deze opdracht is op school gemaakt voor het vak van DevOps gegeven door Quinten Desmyter

Context

Voor deze opdracht heb ik een windows forms gemaakt die moest werken met dapper en een micro ORM zeker. Mijn applicatie gaat de top 10 meest beluisterde liedjes van Apple music in België ophalen. deze worden één voor één getoond, hier kan je zien van wie het liedje is, de naam van het lied en de albumcover. Dit windows formulier heeft 4 knoppen: "Vorige" om naar het vorige liedje te gaan. "Volgende" om naar het volgende liedje te gaan. "Opslaan" om het muzieknummer op te slaan. De opgeslagen muzieknummers worden getoond onder de knop "Opgeslagen muzieknummers" op een volgend schermformulier.

Bijgeleerd

Door dit project heb ik voor het eerst met C# leren werken. In dit project heb ik ook geleerd hoe een micro ORM en een WPF werkt.

Project PHP

Deze opdracht is op school gemaakt voor het vak van PHP gegeven door Patrick Verhaert, Jordi de Roeck en Michael Cloot

Context

Dit project gaat verder op de opdracht van requirements analysis en ook met hetzelfde team als requirements analysis. Ik zal de opdracht in het kort nog eens uitleggen. In deze opdracht is het de bedoeling dat wij een applicatie maken voor mensen met alzheimers en/of een dissociatieve stoornis. Deze webapplicatie gaat vooral draaien rond de patiënt.

Mijn Bijdrage

In dit groepswerk heb ik mij bezig gehouden met het begin te maken voor een agenda waar ze hun dagelijkse routines, taken en afspraken in kunnen plannen en de geassocieerde tabellen in de databank te plaatsen. Hiernaast heb ik ook samen gezeten met iemand anders van mijn groep voor de styling van het project in orde te brengen. Ik ben ook degene die via Combell de website heeft gehost.

Bijgeleerd

Uit dit project heb ik meer ervaring om met een planningsbord te werken om zo te vermijden dat meerdere mensen aan hetzelfde bezig zijn. Als ook heb ik hier uit geleerd om samen aan 1 project te werken met een groep.

Cordova

Deze opdracht is op school gemaakt voor het vak Cordova gegeven door Patrick Verhaert

Context

De opdracht van cordova was een applicatie maken voor op de gsm te zetten. Ik heb hier gekozen om een applicatie te maken die een aantal nieuwsartikelen ophaalt via een API. Deze artikelen kan je filteren op genre en op specifieke woorden die in de titel voorkomen. De opdracht moest ook voldoen aan een aantal eisen en de belangrijkste hiervan was voor een plug-in in de applicatie bij te plaatsen. De plug-in die ik gekozen heb kan connecteren met social media en zo kon ik iets posten via mijn applicatie.

Bijgeleerd

Bij dit project heb ik voor het eerst kennis gemaakt met het framework material, voorheen gebruikte we altijd bootstrap dus dit was ook eens leuk voor mee te werken.
