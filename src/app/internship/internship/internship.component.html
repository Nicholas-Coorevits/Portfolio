Wie is Remmicom

Remmicom is een Belgisch bedrijf opgericht in 1988. Ze zijn gevestigd in het bedrijvenpark van Westmeerbeek, La Corbeille. Ze zijn de specialist in software en hardware oplossingen voor lokale overheden, zowel voor gemeente als OCMW. Ze leveren verschillende diensten zoals:

Remmicom is actief in meer dan 100 administraties over het hele Vlaamse landsgedeelte. Zoals u misschien al wel weet zijn certificaten onmiskenbaar voor overheden. Remmicom is verantwoordelijk voor het beheren van certificaten van talloze organisaties. Concreet zullen wij dus aan de slag gaan met het verbeteren van het huidige proces omtrent het beheren van certificaten.

Opdracht

De stage opdracht was een applicatie ontwerpen die het huidige systeem voor het werken met certificaten kan vervangen. Het huidige systeem vergt veel handmatig werk en heeft geen gebruikersinterface.

Samenvatting

In het begin van de stage gaven ze een een periode van 2 weken die gebruikt konden worden voor het uitvoeren van onderzoek. Tijdens deze fase hadden ik en mijn collega prototypes uitgetekend voor de interface. Hierna was ik begonnen met in visual studio een project op te starten. Waarbij ik heb moeten letten op de benamingen van alles en een correcte project structuur opstellen. Verder had ik een deel van de interface gemaakt namelijk voor certificaten een overzicht te krijgen. Hier had ik ook een zoekbalk opgebouwd die alle data van de voorlopige data in JSON formaat kon sorteren. Eenmaal dit af was ben ik verder gegaan met het back end project te maken.
